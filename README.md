# Linux Cheat Sheet — Beginner to Cloud Engineer

## 1. pwd
Shows the current directory path.

```bash
pwd
```

---

## 2. ls
Lists files and folders.

```bash
ls
```

---

## 3. ls -l
Shows detailed file information and permissions.

```bash
ls -l
```

---

## 4. ls -R
Lists files recursively inside folders.

```bash
ls -R
```

---

## 5. cd
Changes directory.

```bash
cd foldername
```

---

## 6. cd ..
Moves one directory back.

```bash
cd ..
```

---

## 7. cd ~
Moves to home directory.

```bash
cd ~
```

---

## 8. mkdir
Creates a new folder.

```bash
mkdir project
```

---

## 9. touch
Creates an empty file.

```bash
touch notes.txt
```

---

## 10. cp
Copies files or folders.

```bash
cp file1.txt file2.txt
```

---

## 11. mv
Moves or renames files.

```bash
mv old.txt new.txt
```

---

## 12. rm
Deletes files.

```bash
rm notes.txt
```

---

## 13. rmdir
Deletes empty folders.

```bash
rmdir test
```

---

## 14. cat
Displays file contents.

```bash
cat notes.txt
```

---

## 15. less
Views large files page-by-page.

```bash
less notes.txt
```

---

## 16. nano
Opens terminal text editor.

```bash
nano notes.txt
```

---

## 17. echo
Prints text or writes into files.

```bash
echo "Hello Linux"
```

---

## 18. grep
Searches text inside files.

```bash
grep Linux notes.txt
```

---

## 19. find
Finds files and folders.

```bash
find . -name "*.txt"
```

---

## 20. chmod
Changes file permissions.

```bash
chmod 755 script.sh
```

---

## 21. chown
Changes file ownership.

```bash
chown user file.txt
```

---

## 22. ps
Shows running processes.

```bash
ps
```

---

## 23. top
Live system monitor.

```bash
top
```

---

## 24. kill
Stops a running process using PID.

```bash
kill 1234
```

---

## 25. jobs
Shows background jobs.

```bash
jobs
```

---

## 26. systemctl
Manages Linux services.

```bash
systemctl status nginx
```

---

## 27. apt update
Refreshes package lists.

```bash
sudo apt update
```

---

## 28. apt upgrade
Updates installed packages.

```bash
sudo apt upgrade
```

---

## 29. apt install
Installs software packages.

```bash
sudo apt install nginx
```

---

## 30. nginx
Popular Linux web server.

```bash
systemctl status nginx
```

---
